# IFS with Yaniv Rose

A one-page site for 1:1 Internal Family Systems sessions with Yaniv Rose — trauma-informed parts work, online from Chiang Mai, Thailand.

- **Live site:** https://yanivro.github.io/ifs-yaniv-rose/
- **Book a session:** [Calendly](https://calendly.com/yaniv_rose_/one-on-one-yaniv)

## What's here

- `index.html` — the complete, self-contained landing page (all CSS inline)
- `images/` — the ink-and-wash artwork, paper texture, and portrait the page uses

## Editing

Everything is in `index.html`. Open it in a browser to preview locally; push to `main` and GitHub Pages redeploys automatically.
